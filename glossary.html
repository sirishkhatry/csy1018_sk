Terms and their meanings

Cookies-Small text file which includes visit information.CSS- Cascading Style Sheet used to format the look and feel of websites. Domain- The name of website eg. google.com. Hosting- The web servers where your websites files are housed and maintained. HTML- Hypertext Markup Language is used to construct web pages and display contents like text, images etc. Meta Tag- An HTML tag that stores information about a web page. Navigation- Also known as "menu bar" this is navigation element which appear on website. Responsive Design- A website which adjusts depending on what screen it is being view from eg. tablet, phone or laptop. Slider- A rotating banner of images usually placed on a homepage of a website. Wireframe- A visual guide to show the content of a web page without design. UI- User interface means to control the interaction between user and computer with textual/graphical interface. Comment- A bit of information contained in a site's HTML which is ignored by the browser. Compression- This means reducing the size of a web resourse such as video, picture etc. HTML Tags- Pieces of code that are used to identify different parts of HTML document. Hyperlink- A link on a webpage to another web document, usually a webpage or image, sound etc. JavaScript- An object-oriented high level language used for web development.
