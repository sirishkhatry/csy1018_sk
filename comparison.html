Games

This is the Games developer job that i looked at, which is in Bristol, Avon. This particular job pays £22,000-£23,000 per annum. This job is based on HTML5 and JavaScript.Also, they want a experienced developer who has worked in a commercial setting and has a high interest in games development. Also, the applicant must have a BSc in Computer Science or Games Technology. The description of the company is also given which gives information such as the company is a well established games studio which is mainly working on high profile mobile and tablet based games. The main job of the applicant will be to develope multiple games using JavaScript libraries and HTML5.

Web

Web developer jobs have salary of £43,000 - £45,000 per annum, comparing this salary to Games Developer which we looked earlier which was £22,000-£23,000, comparing the incomes game developers earn alot less than web developers. This particular job we are looking at is based on Glasgow, Lanarkshire. For the applicants, their role will be to build large .NET web applications. Strong web design and web development skill is required for this job with flexible and pragmatic attitude. The certain skills required for this job are such as skills in HTML, CSS, Microsoft .NET, C# etc. This Job vacancy has not mentioned any required qualification unlike the games developer one but it seems that varity of skills are needed for this job unlike Games Developer.

Software

Software developers do not require very high skill on HTML compared to Games Developer or Web Developers. The salary also seems to be more than what a Web or Games developer earns which is £45,000-£55,000 per annum This particular job that i have researched on is at Halifax, West Yorkshire. The main skill required for this job is on C# or VB.Net. However, good knowledge of Web services, Automated Unit Testing and Continuous Integration is also required which was not required in any of the jobs that we looked at earlier.
